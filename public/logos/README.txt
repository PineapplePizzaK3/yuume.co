Adicione as imagens das logos das lojas nesta pasta.
Nome do arquivo = id da loja + .png
Tamanho mínimo recomendado: 120x120 pixels.

Logos manuais NÃO são sobrescritos pelo script download-logos.ps1.
O script só baixa logos que ainda não existem. Use -Force para forçar
novo download (ex: .\download-logos.ps1 -Force).
